# 简易 FTP 服务器工具

本项目提供了一个基于 Python 的简易 FTP 服务器脚本，专门用于解决 Windows 环境下的文件共享和中文乱码问题。

## 📂 文件清单

| 文件名 | 描述 |
| :--- | :--- |
| `ftp_server.py` | 一个基于 Python 的简易 FTP 服务器脚本，**已修复 Windows 中文乱码问题**。 |

## 🚀 快速开始

### 简易 FTP 服务器 (`ftp_server.py`)

用于在局域网或公网环境下快速共享文件，并解决了 Windows 资源管理器查看中文文件名乱码的问题。

**依赖安装：**
```bash
pip install pyftpdlib
```

**启动服务器：**
```bash
python ftp_server.py
```

**连接信息：**
*   **地址**: `ftp://<服务器IP>:21`
*   **默认用户**: `admin`
*   **默认密码**: `66666666` (可在脚本中修改)
*   **特性**: 
    *   强制使用 GBK 编码，兼容 Windows 资源管理器。
    *   配置了被动模式端口 (60000-60100)，适合穿越防火墙/NAT。

## ⚠️ 注意事项

*   **FTP 安全**: `ftp_server.py` 默认使用明文传输 (FTP)，建议仅在受信任的网络环境中使用，或仅用于临时文件传输。用完建议关闭。
*   **防火墙**: 如果无法连接 FTP，请检查服务器防火墙是否放行了 TCP 端口 `21` 以及被动端口范围 `60000-60100`。
